# Backend API Project

Ye project PHP aur MySQL par bana hua Backend API hai jo CRUD operations support karta hai.  

## Features
- Add student data (Create)
- Get student data (Read)
- Update student data (Update)
- Delete student data (Delete)
- JSON based API
  -databse is present in name as database.php
## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST   | /insert.php | Insert new student |
| GET    | /get.php    | Get all students |
| PUT    | /update.php | Update student by ID |
| DELETE | /delete.php | Delete student by ID |
